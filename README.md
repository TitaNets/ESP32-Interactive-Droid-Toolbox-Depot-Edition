# Interactive Droid Toolbox – Depot Edition

Interactive Droid Toolbox – Depot Edition is a portable standalone controller and interaction system designed for Disney Galaxy’s Edge Droid Depot droids.

Built around the LilyGO T-Display S3 platform, this project expands the behavior and interaction capabilities of Droid Depot droids beyond the original mobile application experience.

## Supported Droids

- R-Series
- BB-Series
- C1-Series
- CB-23
- D-O
- BD Unit
- A-LT Unit
- R2-H15 Unit

## Supported Personality Chips

- Blue
- Gray
- Red
- Orange
- Purple
- Black
- Yellow
- Navy
- Halloween

## Features

- Portable standalone operation
- USB-C powered
- Compatible with portable battery banks
- Interactive droid behaviors
- Ambient interactions
- Autonomous animations
- Personality chip support
- Charging-friendly operation
- Battery level monitoring
- Multiple interaction modes
- Optimized for degraded droid batteries
- Designed for field use without requiring the original app

## Hardware

Main hardware platform:

- LilyGO T-Display S3
- ESP32-S3
- Integrated TFT display
